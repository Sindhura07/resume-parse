# Resume-parser
# 📄 AI Resume Parser

An AI-powered Resume Parser built using **Python, Streamlit, and Google's Gemini API** to automatically extract and organize candidate information from PDF resumes into structured JSON format.

---

## ✨ Features

* 📂 Upload resumes in PDF format
* 🤖 AI-powered information extraction using Gemini
* 👤 Extract Personal Information
* 🛠️ Identify Key Skills
* 💼 Parse Work Experience
* 🎓 Extract Education Details
* 📜 Capture Certifications and Projects
* 💾 Download structured data as JSON
* 📋 View extracted information in a clean interface

---

## 🖥️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* PyPDF2
* JSON

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Sindhura07/resume-parser
cd ai-resume-parser
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run resume.py
```
